# Unterlagen für Emmetten
In diesem Repo sind die Unterlagen für die ETH-Weiterbildung in Emmetten hochgeladen.

Die benötigte Web-APP ist [hier](https://typst.app).

Die Dokumente können beliebig für eigene Projekte verwendet werden. Das ETH-Template fand ich auf dem öffentlichen [gitlab-Server](https://www.gitlab.ethz.ch/tmoerschell/typst-eth-theme/-/tree/main) der ETH und habe es leicht angepasst.
